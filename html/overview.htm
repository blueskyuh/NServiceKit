Detailed overview of Service Stack


The diagram above shows all the code that is required to write a simple web service in Service Stack.

You only need to write your service logic against strongly-typed DataContract's and Service Stack takes care of hosting and translating your service on all the available endpoints. The endpoint protocols and transport is completely abstracted away (although still discoverable) free-ing you to concentrate

The Xml and Json endpoints are efficient IHttpHandler's that also allow your service to be called via REST HTTP GET requests, i.e. 

/Xml/SyncReply/GetFactorial?ForNumber=2
